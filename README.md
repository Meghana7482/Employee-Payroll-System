# Employee Payroll System

**Tech Stack:** Python, SQLite

## Project Description
Developed a menu-driven payroll management system to store employee details, calculate salaries, and generate reports. Data is stored in an SQLite database.

## Features
- Add, update, delete, and view employee records
- Calculate net salary automatically
- Simple menu-driven user interface

## How to Run
1. Install Python (3.x) and SQLite
2. Run `payroll_system.py`
3. Follow the on-screen menu to interact with the system
